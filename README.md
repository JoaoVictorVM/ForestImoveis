# ForestImoveis

Projeto desenvolvido durante o curso de **Tailwind CSS** da Origamid.  
A aplicação consiste em um **site institucional para uma imobiliária localizada em áreas de floresta**, com foco em apresentar imóveis e informações da empresa de forma clara e moderna.

Todo o layout foi desenvolvido utilizando **Tailwind CSS v4**, permitindo uma construção rápida de componentes e uma estilização consistente ao longo do projeto.

Durante o desenvolvimento do curso, foi possível aprofundar conhecimentos no **ecossistema do Tailwind**, especialmente na criação de **configurações personalizadas e organização de estilos utilitários**.

---

### Tech Stack:

- **HTML**
- **Tailwind CSS v4**

---

### Objetivo do Projeto:

- Praticar a construção de layouts utilizando **Tailwind CSS**
- Entender melhor o funcionamento de **classes utilitárias**
- Explorar **configurações personalizadas do Tailwind**
- Criar um **site institucional responsivo**

---

### Aprendizados:

- Construção de layouts com **Tailwind CSS**
- Organização de estilos utilizando **utility-first**
- Customização de configurações do **Tailwind**

---
